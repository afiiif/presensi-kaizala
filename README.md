<h2 align="center">Presensi Kaizala</h2>

This is a web application to convert Microsoft Kaizala's attendance report from UTC to Indonesian timezone format (WIB, WITA, and WIT). This web application also provides a feature to create attendance report by employee.

## Stacks

:hammer_and_wrench: Build with **Vanilla JS**  
:gear: Task runner: Gulp.js

## Getting Started

Make sure you have installed [Node.js](https://nodejs.org/en/) and [Gulp.js](https://gulpjs.com/).

Install dependencies:
```
npm install
```

You can start editing scss & js files in `src` folder.  
To build css & js, run this command:
```
gulp build
```
